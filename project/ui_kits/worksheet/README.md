# Worksheet — UI Kit (Ages 9–12)

**New product.** A 2-page worksheet that accompanies a JuniorChef recipe, aimed at older kids who've outgrown pure coloring-in. Designed to be printed A4 landscape and completed with a pencil during or after a cooking session.

Each worksheet is paired 1:1 with a recipe and uses the same visual system as the recipe sheet (same logo, same palette, same type). The dish photo and line-art scene from the recipe are reused so the worksheet feels like a continuation, not a new thing.

## Content structure

**Page 1 — Read & understand**
- Vocabulary match (ingredient names ↔ simple definitions)
- Fill-in-the-blanks from the recipe's technique story
- "Where does it come from?" — a tiny world-map marker tying the dish to a country + one cultural fact

**Page 2 — Think & create**
- Scale the recipe: "For 4 → For 6" math tables (multiplication by 1.5 on gram quantities)
- Senses grid: what does the dish *look / smell / taste / sound* like?
- Design-your-own: a blank plate illustration to sketch a variation, plus a "swap one ingredient" prompt

## Fidelity note

This is a **new** product — there is no existing worksheet to copy. The layout is an original design grounded in the existing visual system (same fonts, colors, logo, shadow patterns as the recipe sheet).
